# SUPFILE

SUPFILE est une plateforme de gestion et de stockage de fichiers inspirée des solutions cloud modernes.  
Le projet propose une interface Web intuitive permettant aux utilisateurs de gérer leurs fichiers de manière simple et efficace.

---

## Objectif du projet

Développer une plateforme complète comprenant :

- Un Dashboard avec visualisation de l’espace de stockage
- Un gestionnaire de fichiers (upload, suppression, renommage, déplacement)
- Un système de recherche et de filtrage
- Une gestion des paramètres utilisateur
- Un design system cohérent et moderne

---

## Fonctionnalités principales

### Dashboard
- Visualisation du quota (utilisé / disponible)
- Graphique de répartition par type de fichiers
- Liste des derniers fichiers ajoutés ou modifiés

### File Manager
- Affichage des fichiers et dossiers (table / grille)
- Navigation via breadcrumb
- Actions : renommer, déplacer, supprimer
- Recherche par nom et extension
- Filtres par type et date

### UI / UX
- Design System SUPFILE
- Interface responsive
- Thème clair / sombre avec persistance
- Expérience utilisateur optimisée
